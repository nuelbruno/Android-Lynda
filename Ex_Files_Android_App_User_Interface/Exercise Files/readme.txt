The Android Studio projects in the Exercise Files have been updated 
to work with the stable release of Android Studio 2.3. The projects
now require at least this version of Android Studio, and the following 
versions or later of these support libraries:

com.android.support:appcompat-v7:25.2.0
com.android.support:design:25.2.0
com.android.support.constraint:constraint-layout:1.0.1
